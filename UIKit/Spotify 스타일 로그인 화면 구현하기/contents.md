## Firebase Auth 알아보기
- OAuth란?
    - 사용자 인증 방식에 대한 업계 표준
    - ID/PW를 노출하지 않고 OAuth를 사용하는 업체의 API 접근 권한을 위임 받음
    - 기본 개념
        - 1. **User** - Service Provider에 계정을 가지고 있는 사용자
        - 2. **Consumer** - Service Provider의 API(제공)
